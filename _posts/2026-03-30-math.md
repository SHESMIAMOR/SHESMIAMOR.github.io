下面给你一份可以直接交的“最终版”证明。

---

**题目**
在 (\triangle ABC) 中，记
[
BC=a,\quad CA=b,\quad AB=c.
]
若
[
b\cos C+c\cos B=b\sin B+c\sin C,
]
问：当 (C) 为钝角时，这样的三角形是否存在？

---

# 结论

**存在。**
而且可以构造出一个满足条件的**钝角三角形**。

---

# 证明

由余弦定理，
[
\cos C=\frac{a^2+b^2-c^2}{2ab},\qquad
\cos B=\frac{a^2+c^2-b^2}{2ac}.
]
因此
[
b\cos C+c\cos B
= b\cdot \frac{a^2+b^2-c^2}{2ab}

* c\cdot \frac{a^2+c^2-b^2}{2ac}.
  ]
  化简得
  [
  b\cos C+c\cos B
  =\frac{a^2+b^2-c^2}{2a}+\frac{a^2+c^2-b^2}{2a}
  =\frac{2a^2}{2a}=a.
  ]
  所以题设条件等价于
  [
  a=b\sin B+c\sin C. \tag{1}
  ]

---

再由正弦定理，
[
a=2R\sin A,\qquad b=2R\sin B,\qquad c=2R\sin C,
]
代入 (1)，得到
[
2R\sin A=2R\sin^2B+2R\sin^2C.
]
约去 (2R)，得
[
\sin A=\sin^2B+\sin^2C. \tag{2}
]

---

现在构造一个满足条件且 (C) 为钝角的三角形。
令
[
A=B=t,
]
则
[
C=\pi-2t.
]
把它代入 (2)：
[
\sin t=\sin^2 t+\sin^2(\pi-2t)
=\sin^2 t+\sin^2 2t.
]
又
[
\sin^2 2t=4\sin^2 t\cos^2 t,
]
故
[
\sin t=\sin^2 t+4\sin^2 t\cos^2 t.
]
设
[
x=\sin t\quad (0<x<1),
]
则 (\cos^2 t=1-x^2)，上式化为
[
x=x^2+4x^2(1-x^2)=5x^2-4x^4.
]
整理得
[
4x^4-5x^2+x=0
]
即
[
x(4x^3-5x+1)=0.
]
由于 (x>0)，故
[
4x^3-5x+1=0.
]
分解：
[
4x^3-5x+1=(x-1)(4x^2+4x-1).
]
因为 (0<x<1)，故可取
[
4x^2+4x-1=0.
]
解得
[
x=\frac{\sqrt2-1}{2}.
]
所以
[
\sin t=\frac{\sqrt2-1}{2},
\qquad
t=\arcsin!\left(\frac{\sqrt2-1}{2}\right).
]
于是
[
A=B=\arcsin!\left(\frac{\sqrt2-1}{2}\right),
]
[
C=\pi-2\arcsin!\left(\frac{\sqrt2-1}{2}\right).
]

数值上，
[
A=B\approx 11.95^\circ,\qquad C\approx 156.10^\circ.
]
显然
[
C>90^\circ,
]
所以 (C) 是钝角。

这说明：**满足题设条件且 (C) 为钝角的三角形确实存在。**

---

# 最终答案

[
\boxed{\text{存在}}
]

例如可取
[
A=B=\arcsin!\left(\frac{\sqrt2-1}{2}\right),\qquad
C=\pi-2\arcsin!\left(\frac{\sqrt2-1}{2}\right),
]
此时 (C\approx156.10^\circ) 为钝角，且满足
[
b\cos C+c\cos B=b\sin B+c\sin C.
]

如果你要，我也可以把这份整理成**竞赛风格的极简版**，或者整理成**标准作业书写版**。
